## Запуск проекта:
* Склонировать проект на ваш компьютер с Github с помощью команды:
```
git clone https://github.com/ShaidullovMarsel/Test_Task.git
```

* Установить зависимости с помощью команды:
```
npm i
```
* Запустить проект с помощью команды:
```
gulp
```
